3rd Semester Project

Arduino Home Security System

A smart home safety system that detects dangerous gas leaks using an MQ-5 sensor. When gas is detected it triggers a buzzer alarm, switches the indicator LED from green (safe) to red (danger), drives a servo motor to open a window so gas can escape, and simultaneously pushes a real-time Bluetooth alert to a custom Android app built in Kotlin.

MQ-5 Gas Sensor
Buzzer Alarm
Green LED — Safe
Red LED — Danger
Servo → Opens Window
HC-05 Bluetooth

The system works in two states: Safe mode — the green LED stays on, the servo holds the window closed. Alert mode — triggered when gas PPM crosses the threshold, the system fires all outputs simultaneously: buzzer sounds, red LED blinks, servo rotates to open the window, and HC-05 transmits a serial message to the Android app.

The Arduino firmware reads the MQ-5 analog output every 500ms and compares it to a calibrated threshold. If the reading drops back to safe levels, the system automatically resets — servo closes the window, LED returns to green, and an all-clear is sent to the app.

Companion Android App

Gas Alert Android App

A custom Android app built in Kotlin that acts as the wireless display for the home security system. It pairs with the HC-05 Bluetooth module, listens for incoming serial data, and shows a live gas reading — colour coded green for safe and red for danger — with a push notification when the alarm is triggered.

The app connects to the HC-05 module via Android's Bluetooth API. Once paired, it opens a serial input stream and continuously reads comma-separated values sent by the Arduino. The UI updates in real-time: a large status indicator, a numeric gas-level reading, and a log of recent events.

When the Arduino sends an ALERT message the app shows a full-screen red warning, plays a notification sound, and logs the time of the event. Dismissing the alert tells the Arduino to reset if gas levels have cleared.

Web Development

This Portfolio Website

A personal portfolio built as a single HTML file using HTML5, CSS3, and vanilla JavaScript. Features a responsive dark-themed design, animated circuit grid background, interactive project modals (like the one you're reading now!), scroll-triggered animations, and a fully validated contact form.

Built as part of learning HTML and CSS from the BIT curriculum. The design uses CSS custom properties (variables) so the whole colour scheme can be changed by editing just a few lines. JavaScript handles the scroll animations, tab switching, mobile navigation, and form validation.

All accessibility best practices are followed: skip links, ARIA labels, keyboard navigation, focus indicators, and semantic HTML throughout.
